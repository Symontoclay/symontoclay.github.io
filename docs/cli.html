A command line interface for SymOntoClay.

soc help

Prints help.

soc help
soc h

soc run

Runs target NPC and waits when you write 'exit' for exit.

soc run <NPC file name>
soc run

Arguments

<NPC file name>
File name of target NPC. This argument is optional. Without this argument the target NPC is detected by locators.

soc exit

Ends running NPC.

soc exit

soc new

Creates new NPC in new or existing worldspace.

soc new <NPC name>
soc n <NPC name>

Arguments

<NPC name>
Name of new NPC

For creating NPC in existing worldspace run command new in worldspace directory.

soc version

Prints current version of SymOntoClay.

soc version
soc v
