{
    on Init =>
    {

    op () =>
    {
        'Begin Go' >> @>log;
        'End Go' >> @>log;

back to top

Using inheritance in logic searching.

Read more here about inheritance in SymOntoClay DSL.

For set of facts:

{: can(bird, fly) :}
{: bird(#Alisa_12) :}

The query:

Download
{: can(#Alisa_12, $x) :}

returns:

<yes>
$x = fly

The query:

Download
{: can(#Alisa_12, fly) :}

returns:

<yes>

The query:

Download
{: $z(#Alisa_12, $x) :}

returns:

<yes>
$z = can(bird,fly); $x = fly

The relationship beween concepts '#Alisa_12' and 'fly' is set by inheritance, not by direct fact or rule.
