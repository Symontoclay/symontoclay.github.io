IWorld Interface

Namespace: SymOntoClay.UnityAsset.Core
Assembly: SymOntoClay.UnityAsset.Core

Represents core of a game world.

Properties

EnableLogging

Gets or sets value of enable logging. It alows enable or disable logging or remote connection for whole components synchronously.
EnableRemoteConnection

Gets or sets value of enable remote connection. It alows enable or disable remote connection for whole components synchronously. It doesn't touch local logging.
IsActive

Returns true if execution was started, otherwise returns false.
CurrentImage

Returns image info about last created (saved) image.

Methods

SetSettings(WorldSettings)

Sets general settings into the instance. This method can be run just once in the start of game or application. You can not use the framework before calling this method. Calling other methods before this will provoke exceptions. You can not call the method after disposing the instance.
GetBipedNPC(BipedNPCSettings)

Registers an instance of NPC by passed settings. Returned agent provides communication between game level in Unity3D and logical core.
GetPlayer(PlayerSettings)

Registers an instance of Player by passed settings. Returned agent provides communication between game level in Unity3D and logical core.
GetGameObject(GameObjectSettings)

Registers an instance of game object by passed settings. Returned agent provides communication between game level in Unity3D and logical core.
GetPlace(PlaceSettings)

Registers an instance of place by passed settings. Returned agent provides communication between game level in Unity3D and logical core.
Load(IRunTimeImageInfo)

Loads image by image info.
Load(String)

Loads image by Id.
Load()

Loads last image.
Start()

Starts execution loaded image. If there was not loading image default image will be created from source code.
Stop()

Stops execution.
CreateImage(RunTimeImageSettings)

Creates (saves) image of execured code by settings.
CreateImage()

Creates (saves) image of execured code.
GetImages()

Returns list of all available image info.
DeleteImage(IRunTimeImageInfo)

Deletes image by image info.
DeleteImage(String)

Deletes image by Id.
